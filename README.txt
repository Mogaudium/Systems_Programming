To use the MusicPlayer, please navigate to ChineseSpotify\x64\Debug and run server.exe.
After that navigate to MusicPlayer\MusicPlayer\bin\Debug and run MusicClient.exe.

Inside the window, press "Connect" and you will connect to the server which will send you available files and the client window will show them in listBox.
After that select the song and press "Play" to start audio playback. Press "Pause" button to pause the audio playback.

Use the slider to control the audio playback volume.

Also please note that you will have to change the directory of music files inside the server.cpp to appropriate directory on you machine.
